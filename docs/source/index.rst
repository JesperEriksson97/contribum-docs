Welcome to Contribum Managed Package documentation!
===================================

**Contribum Managed Package** is a second generation managed package (2GP) provided by Axenon AB. The back integrates Contribums API Services with Salesforce.
Contribum is used to monitor and pull data through the swedish population registration API. 

Check out the :doc:`usage` section for further information, including
how to install and setup the package.
